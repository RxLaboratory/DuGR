DuGR Settings

You can access the settings panel with the wrench icon on the bottom line of the panel.

Common settings

The upper part of the panel contains common settings which are the same across all Duduf/RxLab scripts.

  • Use the Language button to change the language of the script.
  • Use the Settings file button to change the location where are stored these settings. You can use the settings file to easily move the settings to another workstation, or to sync it using an app like Syncthing.
  • The Color menu allows you to change the highlight color of the script.
  • The UI Mode menu allows you to choose between three modes:
    • Rookie is a mode where there are more texts, which makes the script easier to understand and to use to get started quickly, but the user interface takes more room.
    • Standard is the recommended mode if you already know how to use the script. Some texts are hidden and replaced by icons, which makes the UI smaller and less intrusive.
    • Expert is the mode you can use if you prefer a very small UI, wihout any text and just icons.
  • The Normal mode and Dev & Debug mode switch should always stay on Normal mode unless: you’ve found a bug, and in this case errors shown by the Debug mode can help us tackle it down, or you’re working on the development of the script and you need some debug information.

DuGR settings

The second part of the panel contains DuGR specific settings.

  • You can choose to allow or deny changes to locked layers.
  • DuGR creates a semi-transparent red frame around the composition to let you know your in isolation mode. You can choose its placement:
    • No Frame at all.
    • Frame Above all layers.
    • Frame Below all layers.
  • You can choose how layers must be hidden:
    • Wireframe will replace layers by a bounding box and a cross.
    • Or you can just hide them.
  • You can hide or show the markers (tags) from the layer:
    • Show tags.
    • Hide tags.
  • You can choose to either automatically lock the layers which are hidden, or leave them as they are.

Other

  • The Check for updates button will check online if a new version of DuGR is available for you to download.
